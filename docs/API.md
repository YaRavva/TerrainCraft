Бесплатные API для генерации 3D-объектов по тексту или изображению
На основе анализа доступных источников на 2025 год, вот несколько бесплатных API (или с бесплатным tier), которые могут быть интегрированы в веб-приложение для генерации 3D-объектов, включая procedural ландшафты с тайлами и бесшовным соединением. Я сосредоточился на тех, что поддерживают text-to-3D (генерация по описанию) и image-to-3D (по изображению), с учетом лимитов, чтобы они подходили для прототипа или небольшого проекта. Эти API часто используют открытые модели (например, на базе diffusion или GAN), и их можно комбинировать с локальной логикой для тайлового соединения (например, через Three.js на фронтенде). Я исключил платные или без бесплатного доступа.

Hugging Face Inference API
Идеально подходит, как вы отметили, с лимитом около 300 запросов в час на бесплатном аккаунте (зависит от модели; для популярных — rate limit 50–300 req/hour, с возможным throttling). Поддерживает модели вроде TripoSR (image-to-3D за секунды), Zero123 или Shap-E (text-to-3D через multi-view generation). Для ландшафтов: можно генерировать отдельные тайлы (дома, реки) и соединять их proceduralно.
Документация: https://huggingface.co/docs/inference-endpoints
Пример использования: Запрос к endpoint с промптом для генерации OBJ/GLTF. Бесплатно для open-source моделей.

Replicate API
Бесплатный tier с начальными кредитами (около 1–5 USD эквивалента в месяц, хватит на 100–500 запросов, в зависимости от модели; no hard hourly limit, но usage-based). Хостит модели для text-to-3D (например, OpenAI Shap-E forks) и image-to-3D (Tripo-like). Подходит для procedural генерации ландшафтов: генерируйте тайлы по частям и соединяйте бесшовно с правилами (neighbor rules).
Документация: https://replicate.com/docs
Limits: Free для тестирования, с переходом на pay-as-you-go после исчерпания кредитов.
Релевантные модели: "lucataco/triposr" для быстрой image-to-3D.

Stability AI API
Бесплатный доступ с лимитом 25–100 генераций в день (или эквивалент кредитов; hourly limit ~20 req). Поддерживает Stable Diffusion variants для 3D (text-to-image-to-3D pipeline, как Stable Zero123). Для ландшафтов: Генерация текстурированных тайлов (парки, реки) с seamless blending через depth maps.
Документация: https://platform.stability.ai/docs
Limits: Free community API с токенами; подходит для некоммерческого использования.
Пример: Интеграция с моделями вроде "stability-ai/stable-3d" для procedural сцен.

Meshy AI API
Бесплатный tier с лимитом 50–200 запросов в месяц (hourly ~10–20, в зависимости от сложности). Специализирован на text-to-3D и image-to-3D, генерирует production-ready модели (включая ландшафты с домами, парками). Поддерживает tile-based generation для бесшовных миров (с compositional rules).
Документация: https://www.meshy.ai/api
Limits: Free для базовых запросов; модели экспортируются в GLB/FBX.

Tripo AI API
Бесплатный tier с лимитом 100–300 запросов в месяц (hourly ~30). Фокусируется на быстром image-to-3D (10 секунд на модель) и text-to-3D. Идеально для тайлов: Генерируйте отдельные элементы (реки, кампусы) и соединяйте в unbounded ландшафт. Поддержка сезонных вариаций через промпты.
Документация: https://www.tripo3d.ai/api
Limits: Free для non-commercial; экспорт в GLB, OBJ.

Alpha3D API
Бесплатный tier с лимитом 50–150 генераций в месяц (hourly ~15). Поддерживает text-to-3D и image-to-3D для game-ready ассетов, включая ландшафты. Бесшовное соединение через API endpoints для modular generation.
Документация: https://www.alpha3d.io/api
Limits: Free для тестирования; подходит для прототипов кампусов в сезонах.

Google Maps Tile API (для 3D-тайлов)
Бесплатный tier с лимитом 200 USD кредитов в месяц (хватит на тысячи запросов; no strict hourly limit). Не чисто generative, но предоставляет photorealistic 3D tiles для ландшафтов (с seamless соединением). Можно комбинировать с generative API для overlay (например, добавить procedural элементы).
Документация: https://developers.google.com/maps/documentation/tile
Limits: Free до определенного usage; идеально для базовых тайловых карт.


Эти API можно интегрировать в backend (Node.js/Python), где по промпту/изображению генерируются тайлы, а соединение реализуется клиент-сайд (например, в Three.js для coherent ландшафта). Для повышения производительности — кэшируйте результаты и используйте batch-запросы.
